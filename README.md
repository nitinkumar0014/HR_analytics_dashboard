# HR_analytics_dashboard

## 🧠 Problem Statement

Employee attrition is a major challenge for organizations. This project analyzes HR data to identify key factors contributing to employee turnover and provides actionable insights to improve retention.
## 🔍 Project Overview

This Power BI dashboard helps HR teams understand:

* Why employees leave
* Which departments are most affected
* Key patterns in salary, age, and experience

## 📊 Key Performance Indicators (KPIs)

* Total Employees
* Attrition Count
* Attrition Rate (%)
* Average Salary
* Average Age
* Average Years at Company


## 📈 Dashboard Features

* Attrition by Department
* Attrition by Job Role
* Attrition by Salary Slab
* Age Group Analysis
* Years at Company vs Attrition
* Gender-wise Attrition

📷 Dashboard Preview


## 🛠 Tools & Technologies

* Power BI
* Excel / CSV
* DAX (Data Analysis Expression)


## 🔍 Key Insights

* Employees with lower salaries show higher attrition
* Sales department has the highest attrition rate
* Majority of employees leave within first 2 years
* Younger employees (age 20–30) are more likely to leave


## 💡 Business Recommendations

* Improve salary structure for high-risk roles
* Enhance employee engagement programs
* Focus on early-stage employee retention strategies
* Conduct regular feedback surveys


## 🚀 How to Use

1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Explore dashboard with filters and slicers
